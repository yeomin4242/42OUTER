Markdown Cheatsheet

이 문서는 Markdown 문법에 대한 간단한 안내서입니다.

제목

# H1
## H2
### H3
#### H4
##### H5
###### H6

강조

*이탤릭*, **볼드**, ~~취소선~~

목록

순서 없는 목록

- 항목1
- 항목2
- 항목3

순서 있는 목록

1. 첫째
2. 둘째
3. 셋째

링크

[링크 텍스트](URL)

이미지

![대체 텍스트](이미지URL)

코드

인라인 `코드`

블록 코드

```
여기에 코드를 넣으세요
```

표

| 헤더1 | 헤더2 |
| ------ | ------ |
| 내용1 | 내용2 |

더 많은 Markdown 문법에 대해서는 Markdown Guide를 참조하세요.
