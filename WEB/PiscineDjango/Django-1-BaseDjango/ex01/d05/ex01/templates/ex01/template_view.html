Template Engine

Django의 기본 템플릿 엔진은 웹 페이지의 구조를 정의하는 강력한 도구입니다. 이 엔진은 다음과 같은 기능을 제공합니다:

  • Blocks: 페이지의 다양한 섹션을 재정의하고 상속받을 수 있습니다.
  • Loops: for ... in 구문을 사용하여 리스트나 딕셔너리 같은 반복 가능한 객체를 순회하며 데이터를 표시할 수 있습니다.
  • If 제어 구조: 조건에 따라 특정 섹션의 표시 여부를 결정할 수 있습니다.
  • Context 변수의 표시: 뷰에서 템플릿으로 전달된 컨텍스트 변수를 사용하여 동적 데이터를 표시할 수 있습니다.
